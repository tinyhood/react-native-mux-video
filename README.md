# react-native-mux-video

A wrapper around react-native-video that integrates Mux Data.

## Installation

```sh
npm install react-native-mux-video
```

## Usage

```js
import MuxVideo from "react-native-mux-video";

// ...

const result = await MuxVideo.multiply(3, 7);
```
